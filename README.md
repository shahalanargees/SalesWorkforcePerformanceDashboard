# 📊 Sales Dashboard - Power BI

An interactive Power BI dashboard developed to analyze sales performance, profit trends, regional sales distribution, product performance, and salesperson contributions.

## Features

* Sales KPI Tracking
* Profit Analysis
* Regional Sales Insights
* Product Performance Analysis
* Salesperson Performance Dashboard
* Interactive Filters

## Tools Used

* Power BI
* DAX
* Data Modeling

## Dashboard Overview

This dashboard provides a comprehensive view of sales and profit performance, helping users identify trends and make data-driven decisions.
